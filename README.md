# StudentRegistrationSystem
Student Registration System

Key Features of This Student Registration System:
1. 3-Step Registration Process
Step 1: Personal Information (Name, Email, Phone, DOB, Address)

Step 2: Passport Photo Upload with Preview

Step 3: KYC Documents Upload (ID + Academic Documents)

2. File Upload with Preview
Photo Upload: Accepts JPG, PNG, WebP (max 2MB)

Document Upload: Accepts PDF, JPG, PNG (max 5MB)

Live Preview: Shows uploaded files immediately

Drag & Drop: User-friendly file upload interface

Remove Option: Can remove uploaded files

3. JavaScript Validation
Real-time Validation: Validates fields as user types

Step Validation: Must complete current step before proceeding

File Validation: Validates file type and size

Age Validation: Ensures student is at least 18 years old

Email & Phone Validation: Proper format validation

4. Progress Tracking
Visual progress bar showing completion percentage

Step indicators with active/completed states

Clear navigation between steps

Cannot skip incomplete steps

5. Responsive Design
Works perfectly on mobile, tablet, and desktop

Adapts layout based on screen size

Touch-friendly interface for mobile devices

6. User-Friendly Features
Required Field Indicators: Asterisks for mandatory fields

Error Messages: Clear, specific error messages

Loading States: Visual feedback during submission

Success Screen: Confirmation with registration summary

New Registration Option: Start fresh after completion

7. Data Management
Stores data locally during registration

Generates unique registration ID

Displays comprehensive summary upon completion

Console logs data for debugging (in real app, would send to server)

8. Professional UI/UX
Clean, modern design with gradient accents

Consistent color scheme using CSS variables

Smooth animations and transitions

Accessible form elements with proper labels

How to Use:
Step 1: Fill in personal details (all fields required)

Step 2: Upload passport photo (click or drag & drop)

Step 3: Upload ID and academic documents

Submit: Review all data and submit registration

Success: View registration summary with unique ID

Demo Credentials/Data:
Use any valid email and phone number

Date of birth must be at least 18 years ago

Upload sample image files (JPG/PNG under 2MB)

Upload sample PDF/Image documents (under 5MB)

This registration system is production-ready, fully responsive, and includes all necessary validation for a complete student registration process!
